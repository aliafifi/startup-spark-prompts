Thanks for your interest in contributing! This project thrives on community input.

## How to Contribute

1. **Fork** this repository
2. **Create a new branch** (`git checkout -b add-new-prompt`)
3. **Add your prompt** to the relevant category folder
4. **Follow the prompt template** (see below)
5. **Submit a Pull Request**

## Prompt Template

Please use this format when adding a new prompt:

### [Prompt Title]

> **Model:** ChatGPT / Claude / Gemini / Any
> **Category:** Marketing / Finance / Strategy / etc.
> **Difficulty:** Beginner / Intermediate / Advanced

**Prompt:**
\```
Your prompt goes here
\```

**Example Output:**
A brief description of what to expect.

---

## Guidelines

- Make sure prompts are tested and actually work
- Keep prompts clear and actionable
- No duplicate prompts
- Be respectful in discussions and reviews
